Traverse all verticies connected to the starting vertex Breadth First Search ——- execute(self, param_dict: Mapping[str, Any]) -> Any:

first, unpacks param_dict into graph and start (and maybe end) If end has not been passed in, returns a set of connected nodes Otherwise, return a shortest path between start and end

Notes

Time Complexity:

Let |V| be the number of verticies Let |E| be the number of edges T(n) = O(|V| + |E|)
