class algocollection.search.binary_search.BinarySearch¶

Bases: algocollection.common.algorithm.Algorithm

An implimentation of the binary search algorithm. .. method:: execute(self, param_dict: Mapping[str, Any]) -> Any:

first, unpacks param_dict into array and key Search the sorted array for a key

If the key is found, return the index in the array

(from 0 to len(array) - 1)

Otherwise return None

Notes

Time Complexity:

Let n = len(array) T(n) = O(log_2(n))
